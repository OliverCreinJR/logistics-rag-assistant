# Инструкция по заполнению формы ТОРГ-2

## Назначение формы

Форма ТОРГ-2 — унифицированный акт об установленном расхождении по количеству и качеству при приёмке товарно-материальных ценностей. Утверждена постановлением Госкомстата РФ от 25.12.1998 № 132.

С 2013 года применение формы не является обязательным, однако она остаётся стандартом де-факто и наиболее удобна для предъявления претензий поставщикам и перевозчикам. Для импортных товаров применяется аналогичная форма ТОРГ-3.

## Когда заполняется

ТОРГ-2 составляется при выявлении расхождений по количеству или качеству в момент приёмки или в процессе разборки груза в течение 5 рабочих дней с даты приёмки.

## Обязательные реквизиты

В шапке акта указываются:

- организация-получатель (полное наименование, адрес, телефон, ИНН);
- наименование структурного подразделения, принимающего товар;
- основание для составления акта (приказ, распоряжение);
- номер и дата акта;
- место приёмки товара (адрес склада).

## Сведения о грузоотправителе и грузополучателе

В разделе указываются:

- наименование и адрес грузоотправителя (поставщика);
- наименование и адрес производителя (если отличается от поставщика);
- наименование и адрес страховщика (при наличии);
- наименование и реквизиты грузополучателя.

## Сведения о товарно-сопроводительных документах

Заполняются:

- номер и дата контракта (договора) на поставку;
- номер и дата счёта-фактуры;
- номер и дата ТТН (транспортной накладной);
- номер и дата железнодорожной накладной (если применимо);
- номера коносаментов, авианакладных (для других видов транспорта).

## Сведения о транспортном средстве

Указываются:

- вид транспорта;
- марка, государственный номер ТС;
- номер вагона, контейнера (для ж/д);
- номер пломбы и её состояние на момент приёмки.

## Описание расхождений

Основная часть акта — табличная. Для каждой позиции указываются:

| Реквизит | Содержание |
|---|---|
| Артикул | Номенклатурный код товара |
| Наименование | Наименование товара |
| Единица измерения | Шт., кг, упак. и т.п. |
| По документам | Количество и стоимость по ТТН |
| Фактически | Фактически принятое количество и стоимость |
| Недостача | Разница (если фактически меньше) |
| Излишки | Разница (если фактически больше) |
| Брак | Количество и стоимость повреждённого товара |

## Заключение комиссии

В заключительной части акта:

- описываются причины расхождений (по версии комиссии);
- указываются способы хранения товара до вынесения решения;
- фиксируются особые условия (порча упаковки, отсутствие пломбы и т.п.);
- приводится перечень приложений (фотографии, копии документов, заключения экспертов).

## Подписи

Акт подписывают:

- председатель комиссии (с указанием должности);
- члены комиссии (не менее двух человек);
- материально ответственное лицо;
- представитель грузоотправителя или незаинтересованной организации;
- представитель грузополучателя.

При отказе одной из сторон от подписи делается соответствующая отметка.

## Куда направляется акт ТОРГ-2

После составления акт направляется:

1. **Один экземпляр** — в бухгалтерию грузополучателя для отражения расхождений.
2. **Один экземпляр** — в отдел претензионной работы для подготовки претензии.
3. **Один экземпляр** — грузоотправителю (в составе претензии).
4. **Один экземпляр** — на склад для приобщения к документам по приёмке.

Скан-копия акта в течение 1 рабочего дня после составления загружается в корпоративную систему документооборота.
